## Installation

```bash
git clone https://github.com/ldrahnik/power-supply-mode-switcher-driver
cd power-supply-mode-switcher-driver
sudo bash ./install.sh
```

## Uninstallation

```bash
sudo ./uninstall.sh
```

## Customization

- AC mode `conf/ac_mode.sh`
- BAT mode `conf/battery_mode.sh`
